#水位无线遥测装置
用单片机、2.4G无线模块NRF24L01、超声波集成探头HC-SR04、LCD1602（用于发射节点数据显示）、红外（接受节点遥控装置）、mini12864（接收节点数据显示）及少量外围器件制作的家用水位无限遥测装置，可用于地下室、蓄水池等场地的水位遥测。
